# The Average Man

An interactive single-page exhibit on Quetelet's *l'homme moyen* and the history of statistical normality.

## Setup

Requires [Node.js](https://nodejs.org/) v18 or later.

```bash
cd average-man
npm install
npm run dev
```

Open `http://localhost:5173` in your browser.

## Build

```bash
npm run build
npm run preview
```

## Stack

- React 18 + TypeScript
- Vite
- Tailwind CSS
- Framer Motion
- Recharts
